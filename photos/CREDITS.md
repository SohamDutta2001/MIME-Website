# Photo credits

Track attribution and license for every photo shipped on the live site.
Placeholders don't need credits. Real photos do.

## Format

```
filename.jpg
  Photographer: Name
  Source: Direct / Unsplash / Pexels / commissioned / café-owned
  License: All rights reserved / CC BY 4.0 / Unsplash / etc.
  Date added: YYYY-MM-DD
  Notes: any context (e.g., "shot at the café, golden hour, model-released")
```
